Electrical Safety
Only qualified personnel, with knowledge about hazards associate with electrical equipment is
allowed to install EasyLogic T150. In general, the installation will be following IEC 61010-1
recommendations i to be compliant with this norm.
The modules must be installed on a metallic DIN Rail which is fixed on a metallic
surface. This metallic surface must have an M4 screw marked with this symbol.
According to the norm IEC 61010-1, the ground of the cabinet or installation must be
connected to this screw.
AB_AC/LPVT is mounted in a metallic enclosure offering protection for isolation faults.

General Description
EasyLogic T150 Platform
EasyLogic T150 – part of the EasyLogic range - is a hardware platform by Schneider Electric
specifically designed for real-time control and energy automation applications. EasyLogic T150 is a
high-technology embedded platform which serves Schneider Electric’s business areas.
On this hardware platform, the Baseline Software Platform is installed. This software is used in
EasyLogic T150, PowerLogic T500 and other Schneider Electric products.
Other features identifying EasyLogic T150 are:
•

A DIN rail is used for the mechanical installation.

•

The communication between the controls units integrated in a distributed system is mainly
established by Ethernet.

•

The terminal blocks for field-connection are completely built into acquisition blocks.

Figure 6 – EasyLogic T150

EasyLogic T150’s design has been optimized to meet the most demanding requirements of
multiple sectors:
•

Safety and reliability requirements for power, gas and water supply, as well as sewage
treatment plants, etc.

•

Compliance with electric safety, electromagnetic compatibility, and environmental standards.

•

Centralized monitoring and control of geographically distributed systems which support
hierarchical data acquisition and sharing networks.

•

Local monitoring and control with data sharing capabilities of plant-distributed equipment.

•

Inrush Current

Inrush current is a high overcurrent drawn by an electrical device when first turned on. Inrush
current function allows not to signal overcurrent when such behavior occurs at the start of a
transformer or of a motor.

AB_AC LPVT will signal inrush current when a current value(s) on a R, S, T or N lines becomes
higher than “Inrush Current Threshold” (ID:64) and return to normal value before “Restraint Time
for Inrush Current” (ID:63). "Inrush current X" will be set to 1 for the line in which inrush conditions
have been detected. X stands for R, S, T or N lines.
“Inrush Current Threshold” is a % of the pickup current (ID:35 for N line and ID:36 for R, S and T
lines). The range of “Inrush Current Threshold” (ID:64) will be 0~99% and its setting unit is 1%.
The range of “Restraint Time for Inrush Current” (ID:63) is 0~300000 ms and its setting unit is 1ms.
With value 0, this functionality is deactivated.

A permanent fault is signaled when, after an over current is detected in any line, absence of
voltage is confirmed in R, S and T during a definite time. This time is configured in "Permanent
Fault Indication Set Time" (ID:58) configuration parameter.
Absence of Voltage is confirmed when voltage values in R, S and T are lower than the value
introduced for "Voltage Absence Threshold " (ID:33) configuration parameter.

becomes higher than a set value (Phase Overcurrent Set Value ID:67 | Neutral Overcurrent Set
Value ID:69) during a set time (Phase Overcurrent Set Time ID:68 | Neutral Overcurrent Set Value
ID:70), “Overcurrent X” will be set on the status indicators. X can be R, S, T or N lines.

“Overcurrent X” events will not be set.
The range of “Overcurrent set time” (Phase Overcurrent Set Time ID:68 | Neutral Overcurrent Set
Value ID:70) will be 0~180000ms and its setting unit is 1ms.

Calibration
The AB_AC LPVT module is calibrated in factory with dedicated, high precision calibration
equipment. Under normal circumstances it will not be necessary to (re)calibrate the equipment.
(Re)calibrate is not recommended since the factory calibration values are lost in the (re)calibration
process.
If really required, signal calibration can be performed in the field with the ITB running in data
acquisition mode. Except for the module under calibration, all the remaining acquisition blocks in
the ITB will continue operating normally during this operation.
AB_AC LPVT module calibration must be performed with a calibrated three-phase generator.
The calibration process will be performed with webApp.
If the user-performed calibration is lost for any particular reason, the LED Mnt will switch on and the
module will need to be recalibrated for proper operation.
